# LaTeX AI Translation Validator

A Python tool to enforce mathematical integrity and structural correctness in AI-translated LaTeX documents. It performs a strict regex-based check on math blocks before optionally using a local LLM for semantic verification.

## Prerequisites

* Python 3.8+
* Ollama (running locally)

## Installation

1. Clone the repository:
   ```
   bash
   # Clone the repo into a folder named 'latex-ai-validator'
   git clone https://github.com/Elkkuboi/Latex-AI-translation-sanity-check-script.git latex-ai-validator
   cd latex-ai-validator 
   ```
